# 裂隙之渊 (Rifts)

一款未来星际题材的游戏，所有关卡都围绕"人性"展开。

## 核心理念

**直面人性、破除人性的矛盾**

在浩瀚的星际裂隙中，玩家将穿越层层关卡，每一关都是对人性的一次深刻探索与挑战。

## 故事背景

在遥远的未来，人类文明已经拓展到星际之间。然而，在探索未知宇宙的过程中，人类发现了神秘的"裂隙"——这些空间裂缝不仅连接着不同的维度，更映照出人性深处的矛盾与挣扎。

## 游戏特色

- 🌌 **未来星际设定**：宏大的宇宙背景，充满未知的星际探索
- 🎭 **人性主题关卡**：每个关卡都围绕人性的不同面向展开
- ⚖️ **矛盾抉择**：在道德与生存、理性与情感之间做出艰难选择
- 🔮 **深度叙事**：通过游戏机制探讨人性的复杂与深刻

## 技术栈

- 游戏引擎：Godot 4
